# Muhammad Fauzi Nugroho — Personal Portfolio

A personal portfolio website showcasing my journey, experience, and projects as a Full-Stack Developer. Built with modern web technologies, featuring smooth animations and a responsive dark-themed UI.

## 🚀 Built With
This portfolio itself is developed using:
- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React

## 💼 Featured Projects
Here are some of the key projects highlighted in this portfolio:
- **PasalPintar.id** - An AI-powered legal assistant web application using Next.js and Vercel AI SDK.
- **Nadi** - A comprehensive HRIS and performance management system with Role-Based Access Control.
- **Sivitas** - A Village Information System (SIVITAS) for digitizing rural administration using Prisma ORM and PostgreSQL.
- **Dispetaru DIY** - UI redesign and SPA implementation for the Dispetaru DIY public portal.
